Key Flows
How the pieces move together
🌱 New User Signup
1
User signs up at /sign-up via Clerk
2
Clerk fires user.created webhook
3
Webhook creates Stripe customer + User in DB
4
Checks for pending GardenInvitations → auto-accepts
5
Middleware detects onboardingComplete=false → redirect to /onboarding
6
WizardShell → completeOnboarding()
7
Creates Garden + Season + Bed in DB transaction
8
Redirect to /garden/[id]
🪴 Assigning a Plant to a Cell
1
User taps empty cell in BedGrid
2
PlantPicker opens in side panel
3
Search calls searchPlantsAction() → Perenual or DB cache
4
User selects plant → assignPlant() fires
5
Creates Planting row in DB
6
createRemindersForPlanting() generates START_SEEDS, TRANSPLANT, HARVEST reminders (future dates only)
7
Spacing check runs → warns if neighbors too close
8
Page revalidates → cell shows plant + status color
🔔 Reminder Delivery
1
Vercel Cron fires hourly → /api/cron/dispatch-reminders
2
Finds reminders with scheduledAt ≤ now, not sent, not dismissed
3
Groups by user; checks if it's 7–9am in user's timezone
4
Checks NotificationPreference for each type
5
Sends email via Resend if channelEmail=true
6
Sends push via web-push if channelPush=true
7
Marks sentAt=now
8
Removes dead push subscriptions (410 response)
❄️ Frost Alert
1
Cron every 3h → /api/cron/refresh-weather
2
Fetches forecast for gardens with active plantings → upserts WeatherCache
3
Daily cron → /api/cron/frost-check
4
Reads WeatherCache; finds any day ≤ 35°F in 72h
5
Creates FROST_ALERT Reminder for owner + editors (deduped)
6
Next reminder dispatch delivers email + push
7
FrostAlert banner also shows live on Calendar page from cache
💳 Upgrading to Pro
1
User clicks upgrade in BillingClient
2
POST to /api/stripe/checkout with priceId
3
Creates Stripe Checkout Session (with 7-day trial if hadTrial=false)
4
User completes payment on Stripe
5
Stripe fires customer.subscription.created
6
Webhook sets subscriptionTier=PRO in DB
7
Pro features unlock on next page load
🤖 Smart Layout (AI)
1
User opens Smart Layout tab in BedGrid (PRO)
2
Builds plant wishlist in SmartLayoutPanel
3
Calls generateLayoutAction()
4
Fetches bed context: sun map, companions, spacing, zone
5
Calls Claude (opus-4-7) with structured prompt
6
Returns row/col/plantId/reasoning assignments
7
Preview shown on bed grid; user reviews reasoning
8
Accept → acceptLayoutAssignments() bulk-creates Plantings
